响应式WebGL模板
================

这个模板提供了完美的跨平台适配：

✅ 电脑端：适中大小，居中显示，保持9:16宽高比
✅ 手机端：全屏显示，完美适配各种屏幕尺寸
✅ 平板端：自动识别并适配
✅ 自动处理屏幕旋转和窗口大小变化

使用方法：
1. 在Unity编辑器中打开 File → Build Settings
2. 选择 WebGL 平台
3. 点击 Player Settings
4. 在 Publishing Settings 中找到 WebGL Template
5. 选择 "ResponsiveTemplate"
6. 正常打包即可

特点：
- 无需每次打包后手动修改HTML
- 自动识别设备类型并适配
- 保持游戏画面比例和质量
- 支持所有现代浏览器
